EducatEd Choice

A Data-Driven School Matching System

Project Overview

EducatEd Choice helps families compare schools using data-driven insights. The project analyzes school-level academic performance, stability, mobility, and school structure to support better school matching recommendations.

Team and Client

Team Members: Pine Nguyen, John, Victoria

Client: Dr. LaDessa Mitchell / FYG Management LLC

Datasets

  • Georgia Department of Education / GOSA Data
  • Georgia Insights Data Portal
  • NCES Public School Data
  • U.S. News School Data
  • Gwinnett County Public Schools

Data Processing and Model

The datasets were cleaned, standardized, and combined into a school-level dataset. Key columns include graduation rate, SAT score, HOPE eligibility, cohort size, mobility rate, mobility count, and discipline percentage. PCA was used to reduce these variables into meaningful matching dimensions.

Result 1: School Matching Map

The PCA map groups schools based on academic performance and stability. This helps translate complex school data into a simple matching view for families.

Result 2: Key Drivers of School Differences

This chart shows that school differences are shaped by academic performance, school structure, and school scale. This supports the matching system because school choice should consider more than test scores alone.

Result 3: Explained Variance

The explained variance chart shows how much information each PCA component captures. PC1 captures the largest pattern, making it useful for comparing schools.
