EducatEd Choice

A Data-Driven School Matching System

Project Overview

EducatEd Choice helps families compare schools using data-driven insights. Instead of looking at one metric at a time, this project combines multiple academic and structural factors to create a clearer picture of school quality.

Team and Client

Team Members: Pine Nguyen, John, Victoria

Datasets

  • Georgia Department of Education / GOSA Data
  • Georgia Insights Data Portal
  • NCES Public School Data
  • U.S. News School Data
  • Gwinnett County Public Schools

Data Processing and Data Model

Data from multiple sources was cleaned and standardized by removing duplicates, handling missing values, and aligning column formats.

The final dataset includes key school-level features such as: graduation rate, SAT total score, HOPE eligibility percentage, cohort size, mobility rate, mobility count, and discipline percentage.

These variables were used in Principal Component Analysis (PCA) to create two main dimensions: academic performance and school stability. These dimensions power the school matching system.

Result 1: School Matching Map

This visualization groups schools based on academic performance and stability. It transforms complex data into an easy-to-understand map that helps families quickly identify strong school matches.

Result 2: Key Drivers of School Differences

This chart shows which factors have the strongest influence on how schools differ. Academic performance, school structure, and school size all contribute to how schools are grouped, supporting a more balanced matching system.

Result 3: Explained Variance

This chart shows how much information each PCA component captures. The first component explains the largest share of differences between schools, making it a strong summary for comparison.

Bonus D3.js Chart: PCA Variance Explained

This chart shows how much information each principal component captures. PC1 explains the largest share, meaning it captures the most important academic performance pattern used in our school matching system.
